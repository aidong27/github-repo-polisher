## Role

## Why does this repository qualify?

GitHub Repo Polisher is a browser-only open-source tool for small maintainers. It audits public GitHub repos for README coverage, setup commands, screenshots, demo links, license, CI, roadmap, and contribution guidance, then gives practical fixes. The repo is real and runnable: Vite/React app, CI, Pages demo, real screenshots, changelog, issues, and a merged feature PR for rule guidance.

## How I would use API credits

I would use API credits to continue practical maintenance: add scoring presets, Markdown/JSON export, before-and-after audit examples, and clearer README patch suggestions. I would also use Codex for small PRs, review support, release notes, docs updates, and regression checks while keeping changes verifiable with lint, build, browser testing, and changelog entries.
