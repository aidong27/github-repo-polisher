# Security Policy

GitHub Repo Polisher is a frontend-only project that analyzes public repository information in the browser.

## Supported Versions

The `main` branch is the currently supported development version.

## Reporting a Security Issue

If you find a security or privacy issue, please open a GitHub issue with a clear description and reproduction steps.

For sensitive reports, avoid posting private information, account details, private repository data, or other personal material in a public issue.

## Privacy Expectations

The project should remain privacy-friendly by default:

- No account login is required for the default demo flow.
- No private repository access is required.
- No tracking or analytics should be added without a clear project discussion.
- Example data should not contain real private project information.

## Maintainer Notes

Security-related changes should include a short explanation in the pull request body and, when relevant, a note in `CHANGELOG.md`.
